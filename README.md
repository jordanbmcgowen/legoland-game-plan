# Jackson's LEGOLAND Game Plan

A complete day-of strategy guide for visiting LEGOLAND California on April 1, 2026 with a 4-year-old (43" tall), optimized around the Silver Fastrack pass.

## Features

- **Interactive Checklists** — Pre-trip prep, night-before tasks, and packing list with clickable checkboxes that persist via localStorage
- **Silver Fastrack Strategy** — Optimized allocation of 6 skip-the-line rides to save ~4 hours of wait time
- **Hour-by-Hour Timeline** — Full day plan from 9 AM arrival through 7 PM close with walking routes and ride order
- **Ride Reference** — All eligible rides with images, wait times, height requirements, and Fastrack vs standby tags
- **Pro Tips** — Insider advice for maximizing a LEGOLAND day with a young child

## Tech

Single-file static site — just `index.html` with embedded CSS and JavaScript. No build step, no dependencies.

- Vanilla HTML/CSS/JS
- Google Fonts (Inter)
- localStorage for checklist persistence
- Mobile responsive
